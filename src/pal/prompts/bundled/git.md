Git commands for version control.

## Available Subcommands

- `$$git add` - Stage files interactively
- `$$git commit` - Create a git commit with conventional format
- `$$git status` - Show formatted repository status
- `$$git diff` - View changes with smart formatting
- `$$git stash` - Manage stashed changes
- `$$git branch` - Manage git branches
- `$$git log` - View commit history with filtering
- `$$git undo` - Safely undo git operations

Use `$$git <subcommand>` to run a specific command.
Use `$$git --help` to see all available subcommands.
